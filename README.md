# Final Capstone Project: Hospital Outcome Predictor

## Synopsis
The goal of this project was create a Hospital Outcome Predictor using the data gathered in the MIMIC-III Clinical Database Demo. This dataset contains health related data for 100 patients who stayed in the intensive care unit. This database consists  of 26 tables, for this project I utilized the patients.csv file, admissions.csv file and labevents.csv file. The main goal was to utilize this information to develop a length of stay and mortality predictor.

## Dataset
As previously stated, three csv files where mainly used from the MIMIC-III Clinical Database Demo.
### 1.) Patients.csv
The columns [subject_id, gender, dob] were kept and the rest were dropped. Subject_id tied to the patient. The date of birth was converted to proper date time format, filtered to remove unrealistic values, and used to calculate the patients age that they were admitted to the hosiptal. Gender value was coverted to assign male to 1 and female to 0.
